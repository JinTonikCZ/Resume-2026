# Resume-2026

Personal resume and portfolio website for Pavel Dikan, built as a static GitHub Pages project.

## Project Purpose

This repository contains a personal resume website based on the current CV structure.  
The page presents:

- contact information
- work experience
- education and training
- selected GitHub projects
- language skills
- technical skills

In addition, the page includes interactive JavaScript features:

- current local time in Prague
- session timer showing how long a visitor stays on the page
- language switcher
- theme switcher

## Features

- Responsive static portfolio / resume website
- Language switcher: English, Czech, German, Russian
- Theme switcher: dark / light
- Current Prague local time display
- Time-on-page session counter
- GitHub, LinkedIn, Telegram, and YouTube links
- CV PDF download button
- Animated background glow
- Clean card-based layout
- GitHub Pages friendly structure
